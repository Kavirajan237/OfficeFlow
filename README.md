# OfficeFlow – AI Workplace Simulation Platform

## Overview
OfficeFlow is a real-time workplace simulation environment designed for training and evaluating autonomous AI agents in office productivity tasks. The environment simulates common workplace activities such as email management, task completion, meeting scheduling, and workflow decision-making.

The system includes an OpenEnv simulation engine, FastAPI server, Streamlit dashboard, database logging, and Docker deployment, creating a realistic environment for AI agent interaction.

---

## Features
- Email management simulation
- Task management with deadlines
- Meeting scheduling simulation
- Reward-based environment
- Agent interaction using step/reset/state
- FastAPI REST API
- Streamlit dashboard UI
- Database logging (SQLite)
- Docker container deployment
- OpenEnv compatible environment
